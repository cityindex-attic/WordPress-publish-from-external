Class PosteditorModal_upload_to_html

Handles uploads to the Post Editor plugin and converts them to html for use in the editor.

Module strictly works by file types. To add a new file type for processing:

  • add the file extension to $this->accepted_files in $this->construct.
  • Then create a private method to handle the data in the format: parse_<span class="php-var">$filextension</span>( <span class="php-var">$data</span> ) This must return the formated data as html for use in the wordpress post editor.
  • add any css to the file POSTEDITOR_DIR /public_html/css/PosteditorModal_upload_to_html.css
  • don't forget to comment your method to phpDocumentator standard ;)
